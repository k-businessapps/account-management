# KrispCall branded Streamlit app. Upsells and Churn

## Setup
1) Create .streamlit/secrets.toml using the template.
2) Install requirements.
3) Run Streamlit.

### Secrets
Put your Mixpanel auth header in:
.streamlit/secrets.toml

[mixpanel]
auth_header = "Basic ..."

### Run
pip install -r requirements.txt
streamlit run app.py

## Branding
Colors match the KrispCall Media Kit palette.
